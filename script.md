대학교 수강신청을 위한 "시간표 조합기" 웹앱을 만들어줘.

## 핵심 기능

### 1. 과목 입력
- 과목명을 입력하고, 해당 과목의 분반 수만큼 시간 정보를 추가할 수 있어야 해
- 각 분반마다:
  - 분반 번호 (예: 001, 002, 003)
  - 요일 선택 (월~금, 복수 선택 가능)
  - 시작 시간 ~ 종료 시간은 1시간단위로 있는데 입력방식이 09:00이아니라 월1,2,3 등의 방식입
-예시: "C프로그래밍" 과목에서 시간이 다음과 같다면
  - 월1,2,3 -> 월 09:00-12:00
  - 월6,7,8 -> 월 14:00-17:00
  - 수5,6,7 -> 수 13:00-16:00
  - 강의실 (선택사항)
- 예시: "자료구조" 과목에 3개 분반이 있으면:
  - 001분반: 월 1,2,3
  - 002분반: 화 5,6,7
  - 003분반: 수 1,2 목 3

### 2. 여러 과목 관리
- 과목을 계속 추가할 수 있어야 함
- 추가한 과목과 분반 목록을 한눈에 볼 수 있어야 함
- 과목/분반 수정 및 삭제 가능

### 3. 시간표 조합 생성
- "조합 생성" 버튼을 누르면:
  - 각 과목에서 1개 분반씩 선택하는 모든 경우의 수 계산
  - 시간이 겹치는 조합은 자동으로 제외
  - 겹치지 않는 유효한 조합만 리스트로 보여주기
- 총 몇 개의 조합이 가능한지 표시

### 4. 시간표 시각화
- 각 조합을 클릭하면 주간 시간표 형태로 시각화
- 월~금 / 09:00~21:00 그리드
- 과목별로 다른 색상으로 표시
- 과목명, 분반, 강의실 정보 표시

### 5. 추가 기능 (있으면 좋음)
- 선호하지 않는 시간대 설정 (예: 아침 9시 수업 제외)
- 공강일 만들기 옵션 (예: 금요일 공강)
- 조합 결과 저장/내보내기
- 점심시간 확보 옵션

## UI/UX 요구사항
- 깔끔하고 직관적인 디자인
- 입력 → 조합 생성 → 결과 확인 흐름이 명확하게
- 모바일에서도 사용 가능한 반응형
- 시간표는 색상으로 구분되어 한눈에 파악 가능
